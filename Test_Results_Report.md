# Test Results Report — JSONPlaceholder API (Apache JMeter)

## 1. Опис

Звіт містить результати тестування REST API `jsonplaceholder.typicode.com`, виконаного за допомогою **Apache JMeter 5.6.3**.  
Тест охоплює основні HTTP-методи (**GET, POST, PUT, PATCH, DELETE**) і спрямований на перевірку стабільності, часу відповіді та коректності статус-кодів.

## 4. Аналіз результатів

1. GET-запити виконувались стабільно, без помилок, із середнім часом відповіді близько 55 мс.  
2. POST, PUT і PATCH запити показали 100% помилок — очікувана поведінка, оскільки публічний API `jsonplaceholder.typicode.com` не зберігає зміни, а лише імітує їх.  
3. DELETE-запити відпрацювали успішно, із середнім часом відповіді близько 204 мс.  
4. Загальний середній час відповіді по всіх запитах — 177 мс, що свідчить про стабільну роботу тестового середовища.  
5. Використання HTTPS забезпечує безпечну передачу даних, але дещо збільшує затримку.

---

## 5. Висновки

- API `jsonplaceholder.typicode.com` стабільно обробляє запити типу **GET** та **DELETE**.  
- Високий відсоток помилок у **POST**, **PUT** і **PATCH** є очікуваним для цього тестового сервісу.  
- Apache JMeter продемонстрував коректну роботу з HTTP-запитами, асертаціями та збором статистики.  
- Тест-план може бути використаний для навчальних цілей і демонстрації функціонального тестування REST API.
