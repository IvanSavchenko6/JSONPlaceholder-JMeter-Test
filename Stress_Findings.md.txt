# Stress Test Findings — JSONPlaceholder API (Apache JMeter)

## 1. Загальна інформація

Тестування проведено з метою визначення меж навантаження, при яких публічний API `jsonplaceholder.typicode.com` перестає стабільно відповідати.  
Для тестування використано **Apache JMeter 5.6.3**.  
Методи тестування: **GET, POST, PUT, PATCH, DELETE.**  
Тривалість одного прогону — **60 секунд**, кількість користувачів — **20**, розгін (Ramp-up) — **5 секунд**.

---

## 2. Параметри навантаження

| Параметр | Значення |
|-----------|-----------|
| Threads (Users) | 20 |
| Ramp-up Period (sec) | 5 |
| Loop Count | Infinite |
| Duration | 60 секунд |
| Server | jsonplaceholder.typicode.com |
| Protocol | HTTPS |

---

## 3. Результати (Summary Report)

| Метод | # Samples | Average (ms) | Error % | Throughput | Висновок |
|--------|------------|---------------|-----------|--------------|------------|
| GET /posts/1 | 20558 | 31 | 4.77% | 263.0/sec | Стабільна робота, незначні помилки при піковому навантаженні. |
| POST /posts | 437 | 134 | 100.00% | 81.4/sec | API не підтримує створення нових постів — усі запити неуспішні. |
| PUT /posts/1 | 318 | 136 | 100.00% | 69.5/sec | Повна відмова — очікувана поведінка тестового API. |
| PATCH /posts/1 | 350 | 133 | 100.00% | 73.2/sec | Повна відмова — очікувана поведінка тестового API. |
| DELETE /posts/1 | 270 | 130 | 6.30% | 65.6/sec | Стабільно, мінімальні помилки під навантаженням. |
| **TOTAL** | **21933** | **38** | **9.59%** | **109.7/sec** | Загалом система ще стабільна при 20 користувачах. |

---

## 4. Аналіз результатів

1. **GET-запити** витримують 20 одночасних користувачів — середній час відповіді 31 мс, лише 4.77% помилок.  
2. **POST, PUT, PATCH** дають 100% помилок — це нормальна реакція демо-API, оскільки воно не виконує реальних змін.  
3. **DELETE** показує стабільність — лише 6.3% помилок при великій кількості одночасних викликів.  
4. Загальний рівень помилок 9.59% вказує, що система ще «жива», але знаходиться поблизу межі стресового навантаження.

---

## 5. Висновки

- Сервіс `jsonplaceholder.typicode.com` витримує стабільне навантаження до **20 одночасних користувачів** для методів **GET** і **DELETE**.  
- Починаючи з 30–40 користувачів, імовірно, відбудеться збільшення часу відповіді та кількості помилок.  
- Методи **POST**, **PUT**, **PATCH** непридатні для навантажувального тестування, оскільки це демо-сервіс без реальної обробки даних.  
- Отримані результати можуть бути використані як орієнтир для базового навантажувального аналізу REST API.
